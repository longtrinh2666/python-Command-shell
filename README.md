## Features
- `cd <path>`: change the current directory.
- `ls` or `dir`: list files and folders.
- `mkdir <name>`: create a new directory.
- `rm <file>`: delete a file.
- `rm -f <folder>`: delete a folder and all its contents.
- `copy <src> <dst>`: copy a file.
- `copy -f <src> <dst>`: copy a folder.
- `edit <file>`: edit the contents of a file directly in the shell.
- `rename <old> <new>` or `rem <old> <new>`: rename a file or folder.
- `type <file>`: display the contents of a file.
- `python`: enter interactive Python shell mode.
- `date` or `time`: display the current date and time.
- `version` or `ver`: show the shell version.
- `help`: list all supported commands.
- `exit`: quit the shell.
